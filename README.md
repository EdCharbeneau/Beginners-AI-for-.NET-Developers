## Repository Structure

The folders are organized by course module and include:

- Markdown notes and lesson guidance
- Start and end project states for hands-on exercises
- Supporting scripts for building projects

## Prerequisites

Before working through the labs, install and configure:

- .NET SDK 8.0 (or newer)
- Git
- Visual Studio Code (or Visual Studio 2022)
- An Azure OpenAI or OpenAI API key for exercises that call LLM services

Recommended VS Code extensions:

- C# Dev Kit
- .NET Install Tool

Basic setup checklist:

1. Clone the repository.
2. Open the repository root in your IDE.
3. Restore dependencies for the section you are working on.
4. Configure secrets or environment variables for AI service credentials as instructed in each module.
